ZB Story — Flutter project context:
Working on ZBTransactionPainter in ZBAccountChart.

DATA:
- zbtransactions populated with zbcoin from hdchannelRotationList
- zbwallets cleaned of zbcoin (derived from transactions)
- walletsByZBCoin(String zbcoin) added to ZBData — derives wallet ids from transactions
- zbfrequencies is the master enumeration:
  0=off, 1=idkm, 2=silence(red), 3=Breath(yellow), 4=Simple(green), 5=COMPLEX(blue), 6=WHITE CAMEL(white)

INTEGRATION FIELD:
- integrationFieldCenter = midpoint of wallet 57 and wallet 20
- Wallets 10 and 34 draw to iCenter (integration field center)
- _shouldSkip: skips intra-integration pairs except 57→20 and anything involving 10 or 34
- swapForICenter: when 10/34 is subWallet, swap so they originate from their own center
- _pointOffsets: merged start/end offsets for 4 special transactions (48.16, 57.20, 22.12, 36.35)

PAINTER (ZBTransactionPainter):
- _shouldSkip(tx): skips integration-to-integration pairs except 57→20 and 10/34 spokes
- _iCenterWallets = [10, 34]
- _pointsToICenter: any tx involving 10 or 34
- _pointOffsets returns named tuple (p1, p2) offsets per transaction
- setWalletColor maps state int to color pair:
  0→transparent, 1→black, 2→red, 3→yellow, 4→green, 5→blue, 6→white

ZBACCOUNTCHART:
- ZBPreset class: asset, tooltip, onTap
- ZBAccountChart accepts List<ZBPreset> presets (default empty)
- Preset buttons rendered as Column in Positioned(top:10, left:10) inside chart Stack

TOGGLE FUNCTIONS (in parent state):
- _resetAllWallets([int state=0]): force-writes all 64 wallets directly via _writeWalletState
- _updateWallets: respects priority — only writes if walletState > current state
- _toggleSimple: _resetAllWallets(4), setCounters(4)
- _toggleBreath: wallets→3, all counters→3, sacral→4
- _toggleComplex: wallets→5, throat/solar/heart→5
- _toggleSilence: wallets→2, throat/self/sacral/root/spleen/solar→2

CHART FUNCTIONS:
- setSimpleChart: _resetAllWallets(3), setCounters(3)  [old: state 4, now confirm]
- setBreathChart: walletsByZBCoin('Breath')→3, all counters→3, sacral→4
- setComplexChart: walletsByZBCoin('COMPLEX')→5, throat/solar/heart→5
- setSilenceChart: walletsByZBCoin('silence')→2, throat/self/sacral/root/spleen/solar→2
